# SuuntoApp Support

Welcome to the SuuntoApp support page for the Wingfoil app.

This repository is dedicated to bug reports, feature requests, and general feedback about the application.

## Reporting a Bug

Before creating a new issue, please check whether the problem has already been reported.

When submitting a bug report, please include as much information as possible:

- Suunto watch model
- Phone model
- iOS or Android version
- Detailed description of the issue
- Steps to reproduce the problem
- Screenshots, if relevant

## Feature Requests

Suggestions and ideas for improving the application are always welcome.

When proposing a new feature, please describe:

- The problem or need
- The feature you'd like to see
- How it would improve the user experience

## Create an Issue

👉 Go to the **Issues** tab and click **New Issue** to submit a bug report, feature request, or question.

Thank you for your feedback and for helping improve SuuntoApp.
